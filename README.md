# Hybrid LEDNet + Swin-UMamba for Skin Lesion Segmentation
  
---

## Overview
This repository contains the implementation of a hybrid deep learning framework that integrates LEDNet and Swin-UMamba for precise skin lesion segmentation on dermoscopic datasets.

---

## Datasets
The following datasets were used:
- **ISIC-2017 and ISIC-2018:** [https://www.isic-archive.com](https://www.isic-archive.com)  
- **Ph2 dataset:** [https://www.fc.up.pt/addi/ph2%20database.html](https://www.fc.up.pt/addi/ph2%20database.html)
